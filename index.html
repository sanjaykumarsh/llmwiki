Build a Compounding Personal Knowledge Base

LLMwiki operates as an intelligent workspace. Instead of standard chat memory or basic search, the agent compiles source documents into structured, interlinked markdown pages that you can explore and browse here.

To get started:

  1. Create or select a Project from the dropdown at the top.
  2. Click the settings icon on the top right to configure your Gemini API Key.
  3. Drop files (PDF, TXT, MD) into the Raw Sources panel on the left.
  4. Click a raw file and select Compile into Wiki in the agent panel.
  5. Browse the generated pages, click links, and view your knowledge network in the Interactive Graph tab!
